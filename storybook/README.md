# Storybook

## Usage

Create a React app.

```console
npx create-react-app storybook --template typescript
cd storybook
```

Initialize Storybook.

```console
npx sb init
npm run storybook
```

Install other packages.

```console
npm install react-bootstrap bootstrap
npm install sass
```

Deploy to gh pages

```console
npm i @storybook/storybook-deployer --save-dev
```
